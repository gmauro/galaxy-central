Why use Galaxy? It is not a database or a browser. It is an analysis medium that enables multiple tools to be applied to existing data in a simple unified way:
  • It handles large genomic datasets with ease. Watch
  • It contains a unique set of tools for manipulation of genomic coordinates. Read and Watch
  • It is the only resource allowing seamless interactive manipulation of multi-genome alignments (including the latest 28-way). Watch
  • It will soon allow handling and analysis of complex variation data.

Galaxy is produced by the Galaxy team, a part of the Center for Comparative Genomics and Bioinformatics (aka BX) at Penn State.
